# Fullstack django and nextjs Project

Fullstack django and nextjs Project

## Prerequisites

- Docker
- Docker Desketop

## Project Setup

1. **Build and run the project in one step**

   ```bash
   docker compose up --build

   ```
## OR

1. **Build the project**

   ```bash
   docker compose build

   ```

2. **Run the project**

   ```bash
   docker compose up

   ```

## Access the Project
   Open http://localhost:3000 to view the project in your browser.
   Open http://127.0.0.1:8000/admin to view the django admin interface in your browser.
